# Analisando-dados-previdenci-rios-no-mundo

	Sistemas de previdência social são um tipo de política pública de caráter quase universal, sendo um dos principais componentes dos sistemas de seguridade social de diversos países ao redor do mundo. Este projeto tem por objetivo examinar, através de bases de dados de instituições como Banco Mundial, OCDE e FMI, as similaridades e divergências de tais sistemas, bem como seu impacto nos países envolvidos. O projeto analisa dois diferentes tipos de dados - fundos de pensão como % do PIB e gasto público com aposentadorias.
        Através da bibliotecas de visualização e manipulação de dados da linguagem Python, como Pandas, Folium e Matplotlib, e com auxilio de ferramentas abertas de mapeamento, como o pretendemos criar uma visualização interativa de um mapa global, traduzindo dados demográficos e socioeconômicos em algo facilmente compreensível a todos os públicos

Descrição gastos:
Esta visualização compara gastos públicos com aposentadoria de países da OCDE e alguns externos, como Brasil e Rússia. É possível perceber o aumento dos gastos na maioria dos países, como o envelhecimento das populações.

Descrição fundos:
Esta visualização exibe o dinheiro administrado por fundos de pensão como porcentagem do PIB. É possível ver fundos expressivos em países como Chile e Países baixos, comparado a fundos baixos, como os do Brasil e China
